                  \item Agent~$2$: $\exists x_a^{2,0},\random{} x_o^{2,0},\exists x_a^{2,1},\random{} x_o^{2,1},\exists x_a^{2,2},\ldots$
                  \item A linearly ordered prefix for all agents?
              \end{itemize}

        \item Explicitly specify dependency sets in DSSAT
